# 역기획서 — 재료·지형·채굴 (Materials, Terrain, Mining)

> 핵심: **문명의 발전 = 재료를 다루는 능력의 발전.** 더 단단한 암석·광석을 캐려면 더 나은 도구·기술이 필요하고, 그게 다음 재료를 열어준다. 림월드/드워프 포트리스의 결.
> 저장은 추상 숫자가 아니라 **창고 구역(stockpile zone)에 실제로 쌓인 재료 더미**로 한다.

## 1. 림월드에서 가져오는 것
| 메커니즘 | 핵심 | 우리 적용 |
|---|---|---|
| 암석 지대(산) | 단단한 암석 타일, 캐야 통과/획득 | `Rock*` 타일(불가통과) → 채굴 시 청크 + 거친 바닥 |
| 암석 종류 | 화강암/석회암/대리석/사암… 용도·난이도 다름 | 종류별 stone, 단단할수록 도구·기술 필요 |
| 광맥 | 암석 속 광석 | `OreIron` 타일 → 채굴 시 철광석 |
| 제련 | 광석 → 금속 | 철광석 → (제련소) → 강철 |
| 채굴 작업 | 곡괭이로 캠, 시간 소요 | 욕구 기반 채굴 액션, 도구/기술 게이트 |
| 창고 구역 | 바닥에 종류별로 쌓기 | stockpile zone에 실제 더미 배치 |

## 2. 재료 분류 (제안)
- **소비재(따로):** 식량, 식사.
- **건축/가공재(haulable 더미 = `ItemStack.resource`):**
  - `wood` — 지금. 나무 성장 **대폭 감속**해 더 의도적인 자원으로.
  - `stoneSandstone`, `stoneLimestone` — 무른 돌(초반, 맨손/간단 도구).
  - `stoneGranite`, `stoneMarble` — 단단한 돌(더 나은 도구/기술 필요).
  - `ironOre` — 채굴엔 도구, 쓰려면 제련.
  - `steel` — 철광석 제련 산물(기존 추상 steel을 여기에 연결). 아파트/고급 건축 게이트 유지.
- 더미는 종류별로 색·아이콘 구분, 창고 구역에 종류별로 쌓임.

## 3. 지형 (암석 지대)
- 새 타일: `RockSandstone`/`RockLimestone`/`RockGranite`/`RockMarble`(불가통과, 채굴 대상), `OreIron`(암석 속 광맥), `RockFloor`(채굴 후 남는 거친 바닥, 보행 가능).
- 월드젠: 암석 지대 클러스터 몇 개(종류는 지역별로), 그 안에 철 광맥 산재. 림월드처럼 산/노두 느낌.
- `moveCost`: Rock*·Ore* = ∞(나무/물처럼), `RockFloor` = 보행. 타일코드로 저장 자동.

## 4. 채굴 + 도구·기술 게이트 (진행의 엔진)
- 채굴 = 욕구 기반 작업(재료가 필요할 때 콜로니가 스스로 캠). Mine 액션, 시간 소요.
- **게이트(난이도 사다리):**
  - T1(시작): 무른 돌(사암/석회암) — 맨손/돌도구.
  - T2: 단단한 돌(화강암/대리석) + 철광석 — **곡괭이(나무+돌 제작)** 또는 기술 필요.
  - T3: 철광석 → **제련소**에서 강철 → 더 좋은 도구 → 더 깊은 채굴.
- 도구/기술은 콜로니 단위 해금(가벼운 tech 플래그 or era 연동). 이게 "재료 다루기 = 문명 발전"의 핵심 루프.

## 5. 창고 구역 (stockpile zone)
- 구역 = 저장으로 지정된 바닥 타일 집합. 그 위에 종류별 `ItemStack` 더미가 **실제로** 놓인다.
- 운반꾼이 생산물을 가장 가까운 빈 창고 칸으로 날라 내려놓고, 소비자는 거기서 인출.
- 추상 `woodStock`/`steel` 숫자를 **구역 내 실제 더미 합**으로 대체(점진적).
- 자율성: 창고 건물 내부 바닥을 기본 stockpile로, 넘치면 콜로니가 구역을 넓힌다.

## 6. 자율성 원칙 ([[needs-driven-emergence]])
플레이어가 "여기 캐라/저장해라" 안 한다. 콜로니가 *필요*에서: 돌이 필요 → 가까운 무른 암석 채굴; 도구가 없으면 먼저 도구 제작; 강철이 필요 → 철 캐고 제련소 짓고 제련. 도구/기술 해금이 자연히 재료 사다리를 오르게 한다.

## 7. 단계 계획 (Slices)
- **Slice 1 — 지형·재료 토대:** 암석 지대 월드젠 + Rock*/Ore*/RockFloor 타일 + stone/ore 자원 종류 + 렌더 + **나무 감속(완료)**. (보이는 림월드 지형.)
- **Slice 2 — 채굴:** 욕구 기반 채굴 → 돌/광석 더미 산출 + 거친 바닥. 도구/기술 게이트(무른↔단단).
- **Slice 3 — 창고 구역:** 지정 저장 타일 + 구역으로 운반/인출, 추상 재고 → 실제 더미(다종).
- **Slice 4 — 제련·강철:** 철광석 → 제련소 → 강철, 기존 강철 게이트 연결. 곡괭이 등 도구 제작.

## 8. 미해결 결정 (제안 → 확인 필요)
- 암석 종류 범위: 우선 **사암·석회암·화강암 + 철광석**(대리석은 후순위 장식).
- 도구/기술 모델: **가벼운 콜로니 tech 플래그**(곡괭이 보유 여부 등) vs era 연동 — 추천: tech 플래그(재료 루프와 직접 연결).
- 빌드 순서: 재료가 있어야 저장도 의미 있으니 **Slice 1(지형·재료) → 2(채굴) → 3(창고구역)** 추천. (사용자는 "창고 먼저" 선호 — 원하면 wood만으로 창고구역 먼저도 가능.)
