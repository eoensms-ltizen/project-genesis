# Roadmap

## Phase 1: Autonomous First Resident

- Generate a 64x64 tile map.
- Add random residents through the UI.
- Let residents find trees, move, chop wood, and select a house site.
- Show state changes in a game log.

## Phase 2: Roads

- Let players draw roads.
- Make residents prefer roads when moving.
- Prefer house sites near roads.

## Phase 3: Survival Loop

- Add hunger, food, farming, and rest as meaningful systems.

## Phase 4: Social Simulation

- Add conversations, affinity, relationships, and families.

## Phase 5: Jobs

- Add builder, farmer, fisher, woodcutter, and hauler roles.

## Phase 6: Settlement Growth

- Add storage, markets, workshops, and simple production chains.
