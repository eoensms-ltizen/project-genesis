# Game Design

## Fantasy

The player is a creator and planner watching residents form the first shape of a village.

## MVP Loop

1. Player adds a resident.
2. Resident enters Idle.
3. Resident searches for a tree.
4. Resident moves to the tree.
5. Resident chops wood.
6. Tree becomes grass.
7. Resident finds a nearby house site.
8. Resident marks the future home.

## Tiles

- Grass: default walkable terrain.
- Tree: resource source.
- Water: blocked terrain.
- Dirt: reserved terrain type.
- Road: reserved for player road drawing.
- HouseSite: selected future building site.

## Resident States

- Idle
- FindTree
- MoveToTree
- ChopTree
- FindHouseSite
- MoveToHouseSite
- PlanHouse
- Rest
